### install screen-extensio
```bash
git clone https://github.com/HenzelMoras/screen-extensio.git
cd screen-extensio
chmod u+x setup.sh
./setup.sh
```

### uninstall screen-extensio
```bash
chmod u+x uninstall.sh
./uninstall.sh
```
